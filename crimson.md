# crimson

## 概要
```
カードを蒐集しながら迷宮を探索する。迷宮踏破遊戯。
あらゆる資源はカード化する。
蒐集者は一人。アバタールでカード一体になる事ができる。
戦闘は、カード１つにつき、スキルが２つあり、いずれかを使う。第一スキル。第二スキル。
スタックして攻撃力と防御力を上げていく。
耐久力はカードの耐久力の累計。迷宮探索時に固定される。

カードは属性と種族とレアリティがある。
扉は、常にカードの能力を使い開ける必要がある。
炎属性で開ける事ができる。
レアリティ２以上で開ける事ができる。

戦闘と探索のバランスを取りながら、迷宮を踏破する必要がある。
同じカードは四枚まで。

探索の重要スキル。
逃走：必ず逃走できる。それ以外は、逃走条件を満たす必要がある。
万能鍵：あらゆる扉を開ける事ができる。封印扉を除く。
雅語鍵：シナリオにより解除できる。その為には、カードを組み込む必要あり。
錬金：このカードを消費し、錬金する。稀に、第一スキルには錬金スキルがつく。
帰還：使用する事で迷宮から帰還できる。

魔法無効：一度だけ魔法ダメージを受けない。
物理無効：一度だけ物理ダメージを受けない。

カードは取得時には、☆。☆は最大３までつく。
スキルが変わる為、ランクを留める事も有用。
基礎は然程変わらないが、耐久は大きく伸びる。

序盤、中盤、終盤で方針が大きく違う。
序盤：初期デッキ数が少ない為、蒐集者のレベルを上げる。レベル＝構築最大数。
中盤：探索のカードを中心に揃える、条件を満たす為に☆を上げる。
終盤：二つ目のスキルを変更できる錬金が登場する。カードを選別する。

成長、星を上げる事。
錬金、二つ目のスキルを変更する事。第二スキルを取り出し、対象の第二スキルに入れる。

複製ライブラリ、迷宮内に端末があると複製できる。手札のみ。
錬金ライブラリ、迷宮内に端末があると錬金スキルをつける事ができる。手札のみ。
```

## レベリング
```
空札を消耗する事でレベルを上げる事ができる。自動では上がらない。
空札は、星と同じ数。
レベルと同じ数だけ消耗する。
レベル１から２は、カードを１枚消耗。
レベル１０から１１は、カードを１０枚消耗。
```
