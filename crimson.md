# crimson

## 概要
```
カードを蒐集しながら迷宮を探索する。迷宮踏破遊戯。
あらゆる資源はカード化する。
蒐集者は一人。アバタールでカード一体になる事ができる。
戦闘は、カード１つにつき、スキルが２つあり、いずれかを使う。
スタックして攻撃力と防御力を上げていく。
耐久力はカードの耐久力の累計。

カードは属性と種族とレアリティがある。
扉は、常にカードの能力を使い開ける必要がある。
炎属性で開ける事ができる。
レアリティ２以上で開ける事ができる。

戦闘と探索のバランスを取りながら、迷宮を踏破する必要がある。
同じカードは四枚まで。

探索の重要スキル。
逃走：必ず逃走できる。それ以外は、逃走条件を満たす必要がある。
万能鍵：あらゆる扉を開ける事ができる。封印扉を除く。
雅語鍵：シナリオにより解除できる。その為には、カードを組み込む必要あり。

カードは取得時には、☆。☆は最大４までつく。
スキルが変わる為、ランクを留める事も有用。同じカードにも反映される。
基礎は然程変わらないが、耐久は大きく伸びる。

```
