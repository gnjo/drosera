## 防御効果
```
防御はDD　Defence and Dodge
物理攻撃における回避率。ダメージの低減効果はない。
敏捷と体力の高い方を参照し、基礎回避率とし、防具を加える。防具１につき、４％加える。
敏捷２０、体力１５の場合、２０を基礎回避率とし、防具８なら、３２加える。
この時の回避率は５２％。命中率は、相手の回避率の反値となる。
状態異常は、最終回避率を半減させる。

魔法防御は、まず抵抗値でダイスを振り、回避できるか判定する。
その後、敏捷か幸運の高い方を参照し、回避できるか判定する。
相手よりレベル上位であれば、もう一度幸運の値で回避できるか判定する。
いずれも失敗した場合、ダメージを抵抗値分減少させ、被弾する。

状態異常防御も魔法防御と同じである。
```
