```

鞄量は、迷宮進入時に固定される。
```

```
フェイ、オオガエル、プリエステス、ヨウコハン、マーチャント

３．立ち去る。

２．戦い、強いる。戦う度に強化される。
３．立ち去る。

プリエステス、対価は宝石。

３．立ち去る。

ヨウコハン、対価はアイテム半分。
１．アイテムを半分渡し、帰還する。

３．立ち去る。
```
