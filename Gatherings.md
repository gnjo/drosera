# Gatherings
```
主人公は竜族。
キャラクタ自由強化のゲーム
レベルが上がるが生命力が１～１０しか増えない。但し、レベルが上がる毎にアイテムを一つ取り込む事ができる。
レベルは比較的簡単に上がる。
クリアレベルは２００から５００程度。
計算の簡略化を行う為、迷宮内では装備変更不可。

どの程度の数値でクリアできるがが不明である為、章のアークを倒す為に能力値の基準を設ける。
条件は二つ。レベルとある能力値以上。満たされると竜門へ向かう事ができる。

重要アイテム。
穿石：竜族に取り込まれたアイテムを一つ取り出す事ができる。比較的頻繁に出る。
印石：召喚枠を１つ拡張する。
魔石：魔技枠を１つ拡張する。
喚石群：戦闘時に常に５体召喚される。

強化には三種類ある。
基本取り込み：あらゆる物を取り込む事ができる。武具記号と部位を取り込む。
喚石取り込み：召喚石を取り込む事ができる。枠は最大１２。別に拡張する必要がある。
魔技取り込み：魔法か技を取り込む事ができる。枠は最大１２。別に拡張する必要がある。


竜族が装備できる武具は、武器、大盾、道具４つのみ。

武器：あらゆる武器を使用する事ができる。
大盾：代わりに武器を装備できるが、大盾は最終ダメージを低減する。
道具１：使用回数が決まっている。迷宮では装備の変更が不可能。
道具２
道具３
道具４
アークは状態異常を付与しなければダメージを与える事が出来ない。

```
## 施設
```
//最少にする事
ガレオンの隅：依頼の掲示がある。
二階角の宿：強化を行う。
御用達の商館：アイテムの売買ができる。
転移門：依頼を受けると、依頼に応じた迷宮へ向かう。依頼を受けない場合は、既にクリアした迷宮を選択。
竜の祠：条件を満たすと行けるようになる。満たさない場合、目的の再提示がある。
```
